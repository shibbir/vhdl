## VHDL
> This repository contains some basic examples of components with VHSIC Hardware Description Language.

## Example components
- NAND Gate
- NOR Gate
- XOR Gate
- RS Flip-flop
- Shift Register
- Elevator FSM
- Full Adder
- Generic Half Adder
- Seven Segment Decoder
