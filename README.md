## VHDL
> This repository contains some basic examples of components with VHSIC Hardware Description Language.

## Example components
- Logic Gates
    - NAND Gate
    - NOR Gate
    - XOR Gate
    - XNOR Gate
- Adders
    - Full Adder
    - Four bit Half Adder
    - Generic Half Adder
    - Sequential Adder
- Flip-flops
    - D Flip-flop
    - RS Flip-flop
- Shift Register
- Elevator FSM
- Seven Segment Decoder
